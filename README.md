## Product Registration System (PHP)
Este foi meu primeiro mini projeto utilizando PHP. A aplicação permite cadastrar produtos, visualizar todos os produtos cadastrados e deletá-los quando necessário.

Os dados são armazenados em um arquivo JSON, funcionando como um pequeno banco de dados para a aplicação. O projeto segue a lógica básica de um CRUD, contendo as operações de **Create**, **Read** e **Delete** (sem a funcionalidade de Update).

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento backend com PHP, manipulação de arquivos e estrutura básica de aplicações web.
