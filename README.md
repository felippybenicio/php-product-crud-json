# 📦 Product Partial CRUD System (PHP + JSON)

A simple product management system built to practice core web development concepts.
This project represents my first hands-on experience developing a partial CRUD application using PHP.

## 🚀 Features

* Create new products
* Display products as cards
* Delete products
* Data persistence using JSON file

## 🛠️ Tech Stack

* PHP (Vanilla)
* HTML5
* CSS3
* JSON (file-based storage)

## 📚 What I Learned

* Partial CRUD operations (Create, Read, Delete)
* Handling and manipulating JSON data in PHP
* Basic backend logic structuring
* Dynamic content rendering
* Initial separation between logic and UI

## 🎯 Purpose

The main goal of this project was to understand how a complete system works — from data input to visual representation.

## 📦 Sistema de CRUD Parcial de Produtos (PHP + JSON)

Um sistema simples de gerenciamento de produtos desenvolvido para praticar conceitos fundamentais de desenvolvimento web.
Este projeto representa minha primeira experiência prática criando um CRUD parcial utilizando PHP.

## 🚀 Funcionalidades

* Criar produtos
* Listar produtos em formato de cards
* Remover produtos
* Persistência de dados em arquivo JSON

## 🛠️ Tecnologias utilizadas

* PHP (puro)
* HTML5
* CSS3
* JSON (armazenamento simples)

## 📚 Conceitos aplicados

* Operações CRUD parciais (Create, Read, Delete)
* Manipulação de JSON com PHP
* Estruturação básica de lógica backend
* Renderização dinâmica
* Separação inicial entre lógica e interface

## 🎯 Objetivo

O objetivo principal foi entender o funcionamento completo de um sistema, desde a entrada de dados até a exibição na interface.

## ⚠️ Limitações

* Não possui funcionalidade de atualização (Update)
* Não utiliza banco de dados
* Sem sistema de autenticação
* Estrutura simples (sem padrão MVC)

## 🔄 O que poderia ser melhorado

* Implementar funcionalidade de atualização (completar o CRUD)
* Substituir JSON por banco de dados (ex: MySQL)
* Adicionar validações e melhor tratamento de erros
* Aplicar um padrão de arquitetura (ex: MVC)
* Melhorar a interface (UI/UX)

## 💡 Sobre o projeto

Mesmo sendo simples, este projeto representa o início da minha jornada como desenvolvedor e a base dos conhecimentos que continuo evoluindo.
